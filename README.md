# TikTok Scraper

A basic tiktok scraper to download videos and comments.

## How to install:

``` bash
git clone git@github.com:AntoinePaix/tiktok-scraper.git
cd tiktok-scraper
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
playwright install
```

## How to use:

```bash
python scraper charlidamelio
```
